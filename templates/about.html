¿Cómo funciona la Calculadora?

Esta calculadora está desarrollada con Flask, un microframework de Python que permite crear aplicaciones web de forma sencilla.

¿Cómo funciona Flask?

Flask maneja las solicitudes web a través de rutas. Cuando el usuario ingresa números y selecciona una operación, Flask procesa la información y devuelve el resultado.

¿Cómo funciona la Calculadora?

  • El usuario ingresa dos números.
  • Selecciona una operación matemática: suma, resta, multiplicación o división.
  • Al presionar "Calcular", Flask recibe los datos, realiza la operación y devuelve el resultado.
